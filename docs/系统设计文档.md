\# 小航AI助手 - 系统设计文档



\## 1. 系统架构

\- \*\*前端/界面层\*\*：Streamlit（st.title, st.selectbox, st.chat\_input）

\- \*\*逻辑层\*\*：Python 函数（身份分流、精准匹配、API调用）

\- \*\*数据层\*\*：Markdown 文件（data/\*.md）

\- \*\*外部服务\*\*：硅基流动 API



\## 2. 模块划分

| 模块 | 文件 | 职责 |

|------|------|------|

| 主界面 | `src/app.py` | 渲染界面，管理session\_state，控制流程 |

| API调用 | `src/api.py` | 发送requests请求，解析返回结果，异常处理 |

| Prompt工程 | `src/prompt.py` | 构建系统提示词，身份分流，防幻觉硬规则 |

| 知识库 | `src/knowledge.py` | 精准匹配QA\_DICT，加载所有md文件 |



\## 3. 交互流程

用户进入 -> 选择身份 -> 点击推荐按钮/输入问题 -> 精准匹配优先（命中直接返回） -> 未命中则拼接Prompt调用API -> 返回并展示回答 -> 保存历史记录。



\## 4. 异常处理设计

\- API超时/断网：捕获异常并提示用户。

\- Key失效：返回401提示。

\- 数据文件缺失：检测目录并提示。



\## 5. 防幻觉机制

\- 系统提示词强制要求“只能根据资料回答”。

\- 资料未收录的必须说“不清楚，拨总值班室”。

\- 回答末尾强制标注【来源】。
