\# 小航AI助手 - 郑航校园信息查询系统



郑州航空工业管理学院 人工智能专业 认知实习项目



\## 技术栈

\- Python 3.10+

\- Streamlit

\- 硅基流动 API (Qwen/Qwen2.5-7B-Instruct)



\## 运行方式

1\. 安装依赖：`pip install -r requirements.txt`

2\. 配置 `src/config.py` 中的 `API\_KEY`

3\. 启动：`streamlit run src/app.py`



\## 功能

\- 身份选择（新生/在校生/教师）

\- 12个推荐问题按钮

\- AI校园问答

\- 电话黄页静态页
